Jeu Cluedo en PHP

Développement d'un jeu de société en ligne avec architecture web complète.

PHP SQLite JavaScript
Voir le code

Description du Projet

Contexte : Projet individuel de BTS SIO consistant à développer une version web du célèbre jeu de société Cluedo, jouable en solo avec une interface web complète.

Objectif : Créer un jeu de déduction où le joueur doit découvrir le meurtrier, l'arme et la pièce du crime à travers un manoir virtuel.

Compétences BTS : B2.1 (Conception et développement d'une solution applicative) et B2.3 (Gestion des données).

Fonctionnalités Implémentées

🎮

Système de Jeu Complet

Mécanique de déduction fidèle au Cluedo original avec personnages, armes et pièces aléatoires

🏠

Navigation Dynamique

Déplacement entre pièces adjacentes avec gestion des portes et restrictions de mouvement

🔍

Hypothèses Intelligentes

Formulation d'hypothèses avec validation et indices progressifs pour guider le joueur

🏆

Gestion de Partie

Suivi des statistiques, fin de partie avec récapitulatif et possibilité de recommencer

📊 Structure de la Base de Données

TABLE pieces (
    id_piece INTEGER PRIMARY KEY,
    nom_piece TEXT,
    description TEXT
)

TABLE personnages (
    id_personnage INTEGER PRIMARY KEY,
    nom_personnage TEXT,
    description TEXT
)

TABLE armes (
    id_arme INTEGER PRIMARY KEY,
    nom_arme TEXT
)

TABLE portes (
    id_piece1 INTEGER,
    id_piece2 INTEGER,
    PRIMARY KEY(id_piece1, id_piece2),
    FOREIGN KEY (id_piece1) REFERENCES pieces(id_piece),
    FOREIGN KEY (id_piece2) REFERENCES pieces(id_piece)
)
